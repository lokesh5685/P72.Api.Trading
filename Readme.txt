-========Application setup & run instructions-=============


Steps:
	1. After codebase download, open P72.Api.Trading.slnx in Visual studio(2026)
	2. Set P72.Api.Trading as start up project, build the application. 
	3. Run the application, swagger UI will be launched in browser.
	4. Use endpoint "GetTradeDetails" to fetch trade details for particular Account Id(e.g. 1, 2)
	5. Use endpoint "GetPositionDetails" to fetch position details for particular Account Id(e.g. 1, 2)




-========End Application setup & run instructions-=========
